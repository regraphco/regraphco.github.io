Regraph

Painless SQL pipeline automation for data warehouses.

Coming soon for Redshift and BigQuery.

No more manual workflow definitions

Regraph scans your code to discover all table definitions and dependencies between them, producing a visualization of your entire pipeline. Changes in code are automatically reflected in your Regraph DAG as your codebase evolves.

Flexible pipeline computation

Run your entire pipeline or pieces of it, one-off or on a schedule. Try a new idea and rapidly propagate changes to downstream tables. Easily make sure tables are up-to-date with all their ancestors.

Instant integration

Manage all your code in git, where it belongs. Regraph pulls the latest changes from Github and deploys them to Amazon Redshift or Google BigQuery, where they can then be accessed in your favorite BI tool. No servers or worker queues to manage, just your code and your data, always in sync.
